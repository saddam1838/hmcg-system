# 🧠 Hierarchical Metacognitive Code Generation (HMCG)

A novel multi-agent framework that unifies **proactive planning**, **collaborative validation**, and **reactive debugging** for robust autonomous code generation. 

This system implements Permutation Invariance (PI) and Permutation Equivariance (PE) checks to ensure structural correctness in multi-agent systems.

## 🚀 Features
- **5-Stage Metacognitive Pipeline:** Planner → Observer → Coder → Validator → Debugger.
- **PI/PE Symmetry Checks:** Validates code for structural symmetry bugs.
- **Auto-Truncation Recovery:** Detects incomplete LLM code and automatically completes it.
- **Modern Streamlit UI:** Real-time progress tracking and interactive validation scorecards.
